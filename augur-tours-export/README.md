# Augur Tours & Travel Export

This folder is an Astro/React export scaffold for Augur Tours & Travel Limited. It includes the source structure, pages, components, API route stubs, data files, configuration files, documentation, and copied public assets needed to continue development as a production booking platform.

## Quick Start

```bash
npm install
cp .env.example .env
npm run dev
```

Open `http://localhost:4321`.

## Included Systems

- Hotel booking with quote requests
- Airport transfers
- Car hire outsourcing requests
- Tour package pages
- Customer dashboard
- Admin dashboards
- Payments module
- WhatsApp, phone, email, SMS, and map integration placeholders
- Gallery, reviews, offers, assistant, and newsletter content modules
