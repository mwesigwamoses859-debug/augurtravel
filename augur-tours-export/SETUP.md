# Setup Instructions

1. Open a terminal in `augur-tours-export`.
2. Run `npm install`.
3. Copy `.env.example` to `.env`.
4. Add real API keys for Google Maps, WhatsApp Business, email, SMS, and payment gateways.
5. Run `npm run dev`.
6. Build for production with `npm run build`.
7. Deploy with Cloudflare using `npm run deploy`.

The current repository includes integration-ready stubs. Live payments, WhatsApp Business, SMS, and email require real provider credentials.
